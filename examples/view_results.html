Active Learning on 3D Probability Simplex

Problem: Learn where f(x) > 0.5 on the probability simplex, where f is the average of 5 smooth step functions.

Method: Active learning with BALD (Bayesian Active Learning by Disagreement) using an ensemble of 5 neural networks.

Results: After 50 queries, achieved 63.14% classification accuracy.

Visualization Panels:

  • Left: True function f(x) values on the simplex
  • Middle: Learned function estimates from the neural network ensemble
  • Right: Classification correctness (TP=green, TN=yellow, FP=orange, FN=red)

Black dots represent the 50 query points selected by BALD acquisition.
